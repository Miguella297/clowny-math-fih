
# Clowny Math Fih

Clowny Math Fih is a development designed to merge both video game and obby fun, with math learning and comprehension. There are 3 main levels of this game, each with a different music piece to help you players progress through each level.

There are as well instructions which can be located on the main screen of the game. Players will use arrow keys to move around and the space key to jump and dodge any obstacles blocking the path of the obby. There are 3 bots on each level which will give the player a math question to answer, enabling to progress throughout the obby if answered correctly.

If at any point the player gets bored with the main levels, there is a mini game (non math related), to keep entertained!
